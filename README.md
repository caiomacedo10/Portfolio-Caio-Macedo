# Portfólio Pessoal – Full Stack Developer

## Sobre o Projeto  
Concluí mais um marco importante na minha jornada como desenvolvedor: finalizei meu portfólio pessoal! Este projeto foi desenvolvido para consolidar meus conhecimentos e apresentar meus principais trabalhos de forma moderna, interativa e clean.

## Tecnologias Utilizadas  

- HTML5 para estruturação semântica  

- CSS3 com design responsivo e moderno  

- JavaScript para adicionar interatividade  

- GitHub Pages para publicação estável e gratuita

## Funcionalidades  

- Design moderno, clean e altamente responsivo, garantindo ótima experiência em qualquer dispositivo  

- Navegação simples e intuitiva para facilitar o acesso aos meus projetos e informações  

- Link direto para WhatsApp, facilitando contato rápido e prático  

- Organização clara dos projetos para mostrar minha evolução e habilidades técnicas

## Desafio e Aprendizado  
Além de aplicar as técnicas de front-end, busquei criar uma experiência agradável e alinhada à minha identidade profissional, reforçando meu posicionamento como Front-End Developer focado em qualidade e usabilidade.
